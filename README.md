## 微信小程序：抖番茄电影

微信小程序抖番茄是一款新闻、电影资讯相关的应用，该项目是使用微信小程序原生语法开发的一款新闻、影评应用，该应用实现了新闻、电影两个模块，完成了热点轮播、详情页、页面跳转、下拉刷新、实时搜索等功能，使用了第三方提供的后端接口实现数据动态更新。
